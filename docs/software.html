Software

The core software maintained by the PSF is as follows:

  • The Cash Stack, as illustrated in the Cash Stack Documentation, from top to bottom:

    • Applications

      • bch-wallet-starter - (demo) - Web wallet, Gatsby Starter
      • gatsby-theme-bch-wallet - Web wallet, Gatsby Theme
      • PSF DEX - Decentralized exchange for SLP tokens.
        • Demo
        • bch-dex - node.js JavaScript REST API server, and production Docker container.
        • bch-dex-ui - Gatsby web-based UI for controlling bch-dex.
        • avax-dex - Avalanche X-chain implementation of bch-dex.
        • avax-dex-ui - Gatsby web-based UI for controlling avax-dex.
      • P2WDB - Pay-to-Write Database
        • p2wdb - Client-side JavaScript library
        • P2WDB Explorer - Developer diagnostic tool (like a block explorer)
        • Developer documentation
      • psf-bch-wallet - Command-line reference wallet.

    • Integration Libraries

      • minimal-slp-wallet - Browser & node.js JavaScript library with basic wallet and token functionality. Has bch-js embedded within it. Can use either web2 or web3 Cash Stack infrastructure.
      • bch-js - Full-featured node.js JavaScript library for crafting custom transactions and doing highly-detailed engineering of the BCH or eCash blockchains.
        • bch-js-examples have code examples for common use-cases.
      • bch-message-lib - Browser & node.js JavaScript library for messaging via the BCH or eCash blockchain and/or IPFS. An extension that requires bch-js.
      • bch-encrypt-lib - Browser & node.js JavaScript library for encrypting messages using the same Eliptic Curve encryption used for Bitcoin Cash and eCash. An extension that requires bch-js.
      • slp-mutable-data - Browser & node.js JavaScript library for creating and reading SLP tokens with mutable and immutable data, following the PS002 specification.

    • REST APIs

      • bch-api - node.js web2 REST API server that integrates the indexer and full node layers. Primary service used by bch-js.
      • ipfs-bch-wallet-consumer - node.js web3 REST API server for minimal-slp-wallet and psf-bch-wallet. Provides a 'local' back end, and proxies requests over IPFS to a 'global' back end via JSON-RPC.
      • ipfs-bch-wallet-service - node.js web3 JSON-RPC server. Receives JSON-RPC commands from ipfs-bch-wallet-consumer over IPFS, then passes the request on to a local instance of bch-api.

    • Indexers

      • psf-slp-indexer - SLP token indexer for BCH and eCash.
      • docker-fulcrum - Docker container for running a Fulcrum Electrumx indexer.

    • Full Nodes

      • docker-bchn - Docker container for a BCHN full node on the BCH blockchain.
      • docker-abc - Docker container for the Bitcoin ABC full node on the eCash blockchain.
